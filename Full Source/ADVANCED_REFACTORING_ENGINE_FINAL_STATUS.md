## 🎓 API Examples

### CLI Usage
```bash
# Extract method
RawrXD-QtShell.exe refactor extract-method main.cpp 10 20 "myMethod" "void"

# Rename symbol
RawrXD-QtShell.exe refactor rename main.cpp oldName newName

# Optimize includes
RawrXD-QtShell.exe refactor optimize-includes main.cpp
```

### Programmatic Usage
```cpp
// Direct API
AdvancedRefactoring engine;
auto result = engine.extractMethod(range, "method", "void");

// CLI Adapter
RefactoringCLIAdapter adapter;
auto json = adapter.executeCommand("extract-method", args);

// Qt Widget
RefactoringWidget widget;
widget.setCurrentFile("main.cpp");
widget.setSelection(10, 20, 0, 0);
```
