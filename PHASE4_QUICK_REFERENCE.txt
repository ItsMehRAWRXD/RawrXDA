CRYPTOGRAPHY REFERENCE

CRC32:         Polynomial-based checksum (integrity check)

═══════════════════════════════════════════════════════════════════════════════
ERROR CODES & MEANINGS
═══════════════════════════════════════════════════════════════════════════════

CACHE_NOT_FOUND (0x0001)        Cache file not found, create new or use online
CACHE_CORRUPTED (0x0002)        Cache file corrupted, delete and sync again
CACHE_EXPIRED (0x0004)          Cache validity expired, connect to network
LICENSE_EXPIRED (0x0008)        License expired, activate new license
GRACE_PERIOD_EXCEEDED (0x0040) License in grace period exceeded, deny features
CLOCK_SKEW_DETECTED (0x0400)   System clock tampered, no offline access
TAMPERING_DETECTED (0x0800)     License key modified, block application
