##  Installation & Setup

### 1. **Start the IDE Completion System**
```bash
# Windows
start-ide-completion.bat

# Linux/macOS
./start-ide-completion.sh
```

### 2. **Install VS Code Extension**
```bash
# The system will automatically package and install the extension
cd n0mn0m-vscode-extension
npm run compile
npx vsce package
```

### 3. **Build IntelliJ Plugin**
```bash
# The system will automatically build the plugin
cd intellij-plugin
./gradlew build
```
