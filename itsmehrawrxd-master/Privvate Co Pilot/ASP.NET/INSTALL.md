#  Quick Installation Guide

## Step 1: Get Your API Key
1. Visit [Google AI Studio](https://makersuite.google.com/app/apikey)
2. Sign in with your Google account
3. Click "Create API Key"
4. Copy the generated key

## Step 2: Load the Extension
1. Open Chrome and go to `chrome://extensions/`
2. Enable "Developer mode" (toggle in top-right corner)
3. Click "Load unpacked"
4. Select this folder: `Privvate Co Pilot/ASP.NET`
5. The extension should appear in your list

## Step 3: Configure
1. Click the extension icon in your browser toolbar
2. Paste your API key in the settings section
3. Click "Save API Key"
4. You're ready to use it!

## Step 4: Create Icons (Optional)
The extension works without custom icons, but for a professional look:
1. Create 3 PNG files: 16x16, 48x48, and 128x128 pixels
2. Name them: `icon16.png`, `icon48.png`, `icon128.png`
3. Place them in the `icons/` folder
4. Reload the extension

## Usage
- **Chat**: Click the extension icon to chat with AI
- **Context Menu**: Select text, right-click, choose " Explain with AI"
- **Smart Detection**: Extension automatically detects code-like content

## Troubleshooting
- **Not working?** Check if API key is saved correctly
- **No context menu?** Refresh the webpage and try again
- **API errors?** Verify your Gemini API key and quota

Enjoy your AI Co Pilot!
