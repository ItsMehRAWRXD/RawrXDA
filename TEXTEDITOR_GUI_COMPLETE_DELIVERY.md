## COMPLETE ARCHITECTURE

### Window Structure
```
Editor Main Window (800x600)
├── Toolbar Window (ToolbarWindow32) - y=0, h=30
│   └── Buttons: Open, Save, Undo, Redo, Cut, Copy, Paste
├── Text Editor Area (y=30, h=540)
│   ├── Line Numbers (left margin, x=0-40)
│   ├── Text Content (x=40-800)
│   └── Cursor (blinking, 500ms timer)
└── Status Bar (msctls_statusbar32) - y=570, h=30
    └── Status Text: "Ready" / "Modified" / "Saved"
```

### Message Routing (WndProc)
```
WinMain/IDE
    ↓
EditorWindow_Show
    ├─ ShowWindow(SW_SHOW)
    ├─ EditorWindow_CreateToolbar()
    ├─ EditorWindow_CreateStatusBar()
    ├─ SetTimer(500ms) ← cursor blink
    └─ Message Loop:
        GetMessageA() →  TranslateMessage() → DispatchMessageA()
            ↓ (WM_*)
        EditorWindow_WndProc() routes to:
            ├─ WM_PAINT (15) → EditorWindow_HandlePaint()
            ├─ WM_KEYDOWN (0x0100) → EditorWindow_HandleKeyDown(12 routes)
            ├─ WM_KEYUP (0x0101) → Track modifiers
            ├─ WM_CHAR (0x0109) → EditorWindow_HandleChar()
            ├─ WM_LBUTTONDOWN (0x0201) → EditorWindow_OnMouseClick()
            ├─ WM_TIMER (0x0113) → Toggle g_cursor_blink
            ├─ WM_DESTROY (2) → PostQuitMessage()
            └─ else → DefWindowProcA()
```

### File I/O Integration
```
File Menu Item (e.g., "Open")
    ↓
ID_FILE_OPEN handler
    ↓
FileDialog_Open() ← GetOpenFileNameA dialog
    ↓ (if selected)
FileIO_OpenRead(filename) ← CreateFileA(GENERIC_READ)
    ↓
FileIO_Read(handle) ← ReadFile to g_file_buffer
    ↓
TextBuffer InsertChar loop ← Copy from buffer
    ↓
FileIO_Close(handle) ← CloseHandle
    ↓
EditorWindow_UpdateStatusBar("Ready")
```

### TextBuffer for AI Integration
```
AI Completion Module
    ↓
Get current line: TextBuffer_GetLineByNum(g_cursor_line)
    ↓
Send to ML model for suggestions
    ↓
For each suggestion token:
    TextBuffer_InsertChar(cursor_pos, token_char)
        ├─ Validate position < 32KB
        ├─ Shift buffer bytes right
        ├─ Insert character
        └─ Return new_size in rax
    ↓
Display completion popup
```
