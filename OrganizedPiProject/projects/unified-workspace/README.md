## 📁 Component Details

### 1. RawrZ Platform

- **Entry Point**: `server.js`

### 2. IDE Suite  
- **Location**: `ide-suite/`
- **Features**: π-Engine, multi-language support, AI integration
- **Tech Stack**: Java, TypeScript, Multi-language execution
- **Entry Point**: `IDEMain.java`

### 3. AI Tools
- **Location**: `ai-tools/`
- **Features**: Agent orchestration, coding assistance, workflow management
- **Tech Stack**: TypeScript, AI providers integration
- **Entry Point**: `src/extension.ts`

## 🚀 Quick Start

### Build All Projects
```bash
# Run master build script
.\build-all.bat
```

### Individual Project Startup
```bash
# RawrZ Platform
cd rawrz-platform
npm install && npm start

# IDE Suite
cd ide-suite  
javac *.java && java IDEMain

# AI Tools
cd ai-tools
npm install && npm run dev
```
