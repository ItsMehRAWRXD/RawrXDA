# RawrXD Text Editor - Quick Reference Card

## STRUCTURES (Keep These Handy!)

### EditorWindow (96 bytes)
```
Offset  Size  Field
0       8     HWND hwnd
8       8     HDC hdc
24      8     Cursor* cursor_ptr
32      8     TextBuffer* buffer_ptr
40      4     uint32_t char_width
44      4     uint32_t char_height
48-95   48    (other fields/reserved)
```

### Cursor (96 bytes)
```
Offset  Size  Field
0       8     uint64_t byte_offset
8       4     uint32_t line_number
12      4     uint32_t column_number
20      8     uint64_t selection_start
28      8     uint64_t selection_end
36-95   60    (reserved)
```

### TextBuffer (2080 bytes)
```
Offset  Size  Field
0       2000  char text_data[2000]
2000    8     uint64_t capacity
2008    8     uint64_t used_length
2016    4     uint32_t line_count
2020    56    line_offsets[7]
```

## CALLING CONVENTIONS (x64 Microsoft)

### Parameter Registers
```
rcx = Parameter 1 (64-bit)
rdx = Parameter 2 (64-bit)
r8  = Parameter 3 (64-bit)
r9  = Parameter 4 (64-bit)
Stack = Parameters 5+
```

### Example Call
```asm
mov rcx, buffer_ptr              ; Param 1
mov rdx, 42                      ; Param 2
mov r8b, 'x'                     ; Param 3 (byte)
call TextBuffer_InsertChar
; Return in rax
```

## CORE FUNCTIONS AT A GLANCE

### Window Setup
```asm
call EditorWindow_RegisterClass       ; Must call first!
mov rcx, &editor_window
mov rdx, L"Title"
call EditorWindow_Create              ; Returns HWND in rax
mov [rcx], rax                        ; Store hwnd
```

### Rendering (Called from WM_PAINT)
```asm
mov rcx, &editor_window
call EditorWindow_HandlePaint          ; Orchestrates all rendering
; Calls: Clear → LineNums → Text → Selection → Cursor
```

### Input Handling
```asm
; Keyboard
mov rcx, &editor_window
mov rdx, VK_CODE
call EditorWindow_HandleKeyDown

; Characters
mov rcx, &editor_window
mov rdx, 'A'
call EditorWindow_HandleChar

; Mouse
mov rcx, &editor_window
mov rdx, x_pixels
mov r8, y_pixels
call EditorWindow_HandleMouseClick
```

### File I/O
```asm
; Open dialog
mov rcx, &editor_window
call EditorWindow_FileOpen             ; Returns filename in rax

; Save to file
mov rcx, &editor_window
mov rdx, filename_string
call EditorWindow_FileSave
```

### AI Token Streaming
```asm
; Single token
mov rcx, &buffer
mov dl, 'x'                           ; Token character
mov r8, &cursor
call Completion_InsertToken

; Multiple tokens
mov rcx, &buffer
mov rdx, "hello world"                ; String pointer
mov r8, &cursor
call Completion_InsertTokenString

; Batch tokens
mov rcx, &buffer
mov rdx, &token_array                 ; Byte array
mov r8, 42                            ; Count
mov r9, &cursor
call Completion_Stream
```

## KEY MEMORY OFFSETS

```asm
; From EditorWindow*
mov rax, [rcx + 0]        ; hwnd
mov rax, [rcx + 8]        ; hdc
mov rdx, [rcx + 24]       ; cursor_ptr
mov rdx, [rcx + 32]       ; buffer_ptr

; From Cursor*
mov rax, [rcx + 0]        ; byte_offset
mov eax, [rcx + 8]        ; line_number
mov eax, [rcx + 12]       ; column_number
mov rdx, [rcx + 20]       ; selection_start
mov rdx, [rcx + 28]       ; selection_end

; From TextBuffer*
lea rax, [rcx + 0]        ; text_data pointer
mov rdx, [rcx + 2008]     ; used_length
mov edx, [rcx + 2016]     ; line_count
```

## FAST DEBUGGING CHECKLIST

**Window not appearing?**
- ✓ EditorWindow_RegisterClass called?
- ✓ CreateWindowExA returning valid HWND?
- ✓ ShowWindow called?

**Text not showing?**
- ✓ TextBuffer_InsertChar updating length at [buffer + 2008]?
- ✓ EditorWindow_RenderText looping through text_data?
- ✓ TextOutA called with correct coordinates?

**Keyboard not working?**
- ✓ WM_KEYDOWN reaching EditorWindow_HandleKeyDown?
- ✓ VK_CODE matching VK_LEFT, VK_RIGHT constants?
- ✓ InvalidateRect triggering WM_PAINT?

**Cursor not blinking?**
- ✓ SetTimer registered in EditorWindow_Create?
- ✓ Cursor_GetBlink returning correct on/off state?
- ✓ EditorWindow_RenderCursor called from HandlePaint?

**File operations failing?**
- ✓ GetOpenFileNameA returning valid filename?
- ✓ CreateFileA returning valid file handle (not INVALID_HANDLE_VALUE)?
- ✓ File permissions allowing write access?

## WNDPROC MESSAGE ROUTING

```asm
EditorWindow_WndProc:
    cmp edx, 1              ; WM_CREATE
    je .OnCreate
    
    cmp edx, 15             ; WM_PAINT
    je .OnPaint → EditorWindow_HandlePaint
    
    cmp edx, 256            ; WM_KEYDOWN
    je .OnKeyDown → EditorWindow_HandleKeyDown
    
    cmp edx, 258            ; WM_CHAR
    je .OnChar → EditorWindow_HandleChar
    
    cmp edx, 273            ; WM_COMMAND
    je .OnCommand → Route to File/Edit handlers
    
    cmp edx, 258            ; WM_TIMER
    je .OnTimer → InvalidateRect (cursor blink)
    
    cmp edx, 2              ; WM_DESTROY
    je .OnDestroy → Release resources
    
    jmp DefWindowProc
```

## COMMON GOTCHAS

🔴 **DON'T**
- Don't forget EditorWindow_RegisterClass before CreateWindowExA
- Don't forget to link against kernel32.lib, user32.lib, gdi32.lib
- Don't use 32-bit registers for pointers (use 64-bit: rax not eax)
- Don't exceed 2000 bytes in text buffer (fixed size)
- Don't call assembly functions from Python directly without C wrapper

✅ **DO**
- Call EditorWindow_RegisterClass first
- Use FRAME prologue/ENDPROLOG for proper x64 discipline
- Pass structures by reference (pointer in rcx/rdx/r8/r9)
- Check return values (especially file operations)
- Use C/C++ wrapper class for IDE integration
