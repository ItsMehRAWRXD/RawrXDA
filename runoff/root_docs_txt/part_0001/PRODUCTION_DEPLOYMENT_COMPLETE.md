### Example 2: CI/CD Pipeline Integration

```powershell
# Start CI/CD monitoring in background
$ciJob = Start-Job -ScriptBlock {
    Import-Module C:/RawrXD/auto_generated_methods/RawrXD.*.psm1
    Invoke-ContinuousIntegrationTrigger -WatchDir 'C:/DevOpsProject' -EnableWebhooks -WebhookPort 8765

# Monitor build queue

    $stats = Get-CIStatistics
    Write-Host "Queue Length: $($stats.QueueLength)"
    Write-Host "Active Builds: $($stats.ActiveBuilds)"
    Start-Sleep -Seconds 30
}

# Get build results
$buildResult = Receive-Job $ciJob
$ciJob | Remove-Job
```

### Example 3: Performance Monitoring

```powershell
# Start metrics collection
```
