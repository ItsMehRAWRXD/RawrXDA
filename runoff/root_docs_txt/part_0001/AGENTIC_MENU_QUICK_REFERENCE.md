# Agentic Menu Quick Reference

## Access All Agentic Features

### Via Menu Bar
```
MenuBar → Agentic → [Choose Feature]
```

### Via Command Palette
```
Ctrl+Shift+P → Type "Agentic" → [Select Command]
```

### Via Keyboard Shortcut
```
Ctrl+Shift+D = Discovery Dashboard
```

---

## 5 Agentic Features Available

### 1. 📊 Discovery Dashboard (Ctrl+Shift+D)
**What It Does:** Real-time monitoring of all autonomous capabilities
**Shows:** Capability cards, performance metrics, activity feed
**Use When:** You want overview of all agentic systems

### 2. 🧠 Planning Engine
**What It Does:** Monitors task decomposition and planning
**Shows:** Plan creation, task breakdown, execution progress
**Use When:** Creating complex project plans

### 3. 🔍 Error Analysis
**What It Does:** Monitors error diagnosis and fix generation
**Shows:** Error analysis, fix suggestions, application results
**Use When:** Debugging compilation or runtime errors

### 4. 🔧 Refactoring Engine
**What It Does:** Monitors code improvement suggestions
**Shows:** Refactoring suggestions, performance issues, code smells
**Use When:** Optimizing or improving existing code

### 5. 💾 Memory Persistence
**What It Does:** Monitors context saving and restoration
**Shows:** Snapshot saves, session restoration, memory optimization
**Use When:** Saving/restoring work sessions

## Quick Actions

### Enable All Systems
```
Menu: Agentic → Enable All Agentic Systems
Palette: Ctrl+Shift+P → "Enable All"
Result: Opens all 5 monitoring panels
```

### Disable All Systems
```
Menu: Agentic → Disable All Agentic Systems
Palette: Ctrl+Shift+P → "Disable All"
Result: Closes all 5 monitoring panels
```

## Panel Operations

### Show/Hide Panel
- **Menu:** Click checkbox next to panel name
- **Palette:** Ctrl+Shift+P → "Show [Panel Name]"
- **Panel X:** Click X button on panel title bar

### Dock Panel
1. Drag panel title bar to edge of window
2. Blue highlight shows dock zone
3. Release to dock panel

### Float Panel
1. Double-click panel title bar
2. OR drag panel away from edges
3. Panel becomes floating window

### Tab Panels Together
1. Drag one panel's title bar
2. Drop onto another panel's title bar
3. Panels become tabbed group

## Example Workflows

### Debugging Workflow
1. Open Error Analysis panel
2. Compile project (build errors appear)
3. Watch Error Analysis auto-diagnose issues
4. See suggested fixes in real-time
5. Apply fixes from suggestions

### Planning Workflow
1. Open Planning Engine panel
2. Use Command Palette: "Create Master Plan"
3. Enter your goal
4. Watch plan generation in real-time
5. See task decomposition and dependencies

### Coding Workflow
1. Open Refactoring Engine panel
2. Edit code in main editor
3. Watch refactoring suggestions appear
4. See performance issues highlighted
5. Apply suggested improvements

### Session Management
1. Open Memory Persistence panel
2. Work on project
3. Use "Save Memory Snapshot" to save state
4. Later: Restore snapshot to continue
5. Watch restoration process in panel

## Tips & Tricks

### Tip 1: Keep Dashboard Open
The Discovery Dashboard provides a bird's-eye view of all systems. Keep it open in a tab group with other panels for easy monitoring.

### Tip 2: Dock Right for Monitoring
Dock all agentic panels to the right edge. They won't interfere with code editing on the left.

### Tip 3: Use Tab Groups
Group related panels: Planning + Error Analysis in one tab, Refactoring + Memory in another.

### Tip 4: Keyboard Efficiency
Use Ctrl+Shift+P to quickly toggle panels without touching the mouse.

### Tip 5: Status Bar Feedback
Watch the status bar (bottom of window) for confirmation messages when toggling systems.

## Troubleshooting

### Panel Won't Open
**Solution:** Use "Enable All" to initialize all panels, then close unwanted ones.

### Panel Disappeared
**Solution:** Check View menu or Agentic menu - uncheck and recheck panel name.

### Too Many Panels
**Solution:** Use "Disable All" to reset, then selectively open needed panels.

### Panel Stuck
**Solution:** Close panel and reopen. Dock widgets auto-reset positions.
