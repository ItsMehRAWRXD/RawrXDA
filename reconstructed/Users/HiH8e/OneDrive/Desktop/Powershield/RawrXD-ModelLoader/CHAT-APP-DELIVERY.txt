## 📋 Integration Guide

### Adding to Build

1. **Files Already Added to CMakeLists.txt:**
   ```cmake
   add_executable(RawrXD-Chat
       src/win32app/main_chat.cpp
       src/win32app/Win32ChatApp.cpp
       src/win32app/Win32ChatApp.h
       src/win32app/ContextManager.h
       src/win32app/ModelConnection.h
   )
   ```

2. **Libraries Already Linked:**
   ```cmake
   target_link_libraries(RawrXD-Chat PRIVATE
       comctl32
       shell32
       winhttp
   )
   ```

3. **Build Command:**
   ```bash
   cd RawrXD-ModelLoader
   cmake -B build -DCMAKE_BUILD_TYPE=Release
   cmake --build build --config Release
   ```

4. **Output:**
   ```
   build/bin/Release/RawrXD-Chat.exe
   ```

### Running the Application

```bash
# Make sure Ollama is running
ollama serve

# In another terminal, run the chat app
./build/bin/Release/RawrXD-Chat.exe
```
