⚙️ HOW IT WORKS
═══════════════════════════════════════════════════════════════════════════════

FUNCTION CALL FLOW:

1. User Input
   └─ Types natural language in RawrXD IDE

2. AI Processing
   └─ bg-ide-agentic model generates response

3. Function Call Detection
   └─ RawrXD parser detects: {{function:tool_name(args)}}

4. Argument Parsing
   └─ Extracts tool name and parameters
   └─ Converts to PowerShell hashtable

5. Tool Execution
   └─ Invokes registered tool via Invoke-AgentTool
   └─ Executes PowerShell command/function

6. Result Capture
   └─ Formats output as JSON
   └─ Handles errors gracefully

7. Display
   └─ Shows result in IDE chat
   └─ Continues conversation context

8. Multi-Step
   └─ AI can chain multiple tool calls
   └─ Tools build on previous results
