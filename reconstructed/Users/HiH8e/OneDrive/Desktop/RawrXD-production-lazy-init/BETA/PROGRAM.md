# RawrXD Beta Program

Welcome to the RawrXD Win32 IDE Beta Program!

## Overview

We're seeking 10-20 beta testers to help validate and improve the RawrXD Win32 IDE before public release.

## What You'll Get

- **Early Access** to cutting-edge AI-powered IDE features
- **Direct Input** on feature development and priorities
- **Priority Support** via dedicated Discord channel
- **Recognition** in release credits

## What We Need From You

### Time Commitment
- **2-4 hours/week** of testing
- **1 feedback session** per week (async or 30-min call)
- **Bug reports** for any issues encountered

### Technical Requirements
- Windows 10/11 (x64)
- 8GB+ RAM recommended
- Willing to install development tools
- Basic familiarity with IDEs

### Testing Focus Areas

1. **Core Functionality** (Week 1-2)
   - File tree navigation
   - Code editing and syntax highlighting
   - Terminal operations
   - Basic UI responsiveness

2. **AI Features** (Week 2-3)
   - Chat interface and streaming responses
   - Code generation with `/cursor` command
   - Image generation with `/paintgen`
   - Orchestrator multi-step execution

3. **Performance** (Week 3-4)
   - Startup time on your hardware
   - Memory usage under heavy load
   - UI responsiveness with many tabs
   - Large file handling

4. **Integration** (Week 4)
   - LSP integration (hover, definitions)
   - Paint canvas and tools
   - Voice features
   - Enterprise logging

## Bug Reporting Guidelines

### Critical Bugs (Report Immediately)
- IDE crashes or hangs
- Data loss or corruption
- Security vulnerabilities

### High Priority (Report Within 24 Hours)
- Feature completely broken
- Significant performance degradation
- UI unusable or blocked

### Medium/Low Priority (Report Weekly)
- UI inconsistencies
- Minor performance issues
- Feature enhancements

### Report Format
```markdown
**Title:** Brief description

**Environment:**
- OS: Windows 11 23H2
- IDE Version: 1.0.0-beta.1
- Hardware: Intel i7-12700, 16GB RAM

**Steps to Reproduce:**
1. Open IDE
2. Create new file
3. Type "test"
4. Crash occurs

**Expected Behavior:**
IDE should not crash

**Actual Behavior:**
IDE crashes with error dialog

**Logs/Screenshots:**
[Attach logs from %LOCALAPPDATA%\RawrXD\logs\]
```
