RawrXD v0.1.0 MVP
==================

Local AI code completion with streaming ghost text.

QUICK START
-----------

1. Download a GGUF model:
   Place it in: models/
   Example: Mistral-7B-Instruct-v0.1.Q4_K_M.gguf

2. Open TWO terminal windows:

   Terminal 1 (Engine):
   cd engine
   run_engine.bat ..\models\Mistral-7B-Instruct-v0.1.Q4_K_M.gguf

   Terminal 2 (IDE):
   cd ide
   run_ide.bat

3. Open browser:
   http://localhost:5173

4. Start typing code
   Ghost text appears in real-time

TROUBLESHOOTING
---------------

Q: Engine crashes on startup
A: Check model path is correct
   Models must be in GGUF format
   Example: run_engine.bat ..\models\model.gguf

Q: IDE won't start
A: Install Node.js from https://nodejs.org/
   Then try: npm install && npm run dev

Q: Ghost text doesn't appear
A: Ensure engine is running on http://localhost:8080
   Check browser console for errors (F12 in browser)
   Model must be fully loaded (check engine window)

Q: Port already in use
A: Engine uses 8080, IDE uses 5173
   Close other applications using these ports
   Or edit run_engine.bat to use different --port
