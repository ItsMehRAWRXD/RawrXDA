🚀 Quick Reference

Build Commands

# CLI IDE
cl /EHsc /O2 /Fe:codex_cli_ide.exe codex_cli_ide.cpp

# GUI IDE (Win32)
cl /EHsc /O2 /Fe:codex_gui_ide.exe codex_gui_ide.cpp ^
  user32.lib gdi32.lib comdlg32.lib shell32.lib comctl32.lib

# ASM Engine (MASM64)
ml64 /c /Fo:codex.obj CodexAIReverseEngine.asm
link /SUBSYSTEM:CONSOLE codex.obj kernel32.lib user32.lib

# Or use the batch script:
build_all_ide.bat

CLI Commands

codex_cli_ide load model.gguf    # Brute-force load
codex_cli_ide scan C:\models\    # Batch scan
codex_cli_ide                    # Interactive REPL

REPL Commands:
  load <path>       Load model
  scan <dir>        Batch scan directory
  quants            List 36 quant types
  tokenizers        List 5 tokenizer types
  archs             List 34 architectures
  entropy <file>    Shannon entropy
  pe <file>         PE binary analysis
  report            Last analysis report
  help | exit
