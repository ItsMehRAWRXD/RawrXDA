HOW TO USE IT RIGHT NOW (WORKING VERSION)
================================================================================

1. OPEN & EDIT FILES:
   - Click on folder in left pane to expand
   - Double-click file to open in editor
   - Edit text
   - Ctrl+S to save

2. RUN POWERSHELL COMMANDS:
   - Ctrl+` (backtick) to focus terminal pane
   - Type PowerShell commands: dir, cd, python, node, etc.
   - Press ENTER to execute
   - See output immediately

3. TRY THESE COMMANDS IN TERMINAL:
   - dir (list files in current directory)
   - cd d:\ (change directory)
   - python --version (if Python installed)
   - node --version (if Node installed)
   - powershell commands work
   - batch/cmd commands work

4. EDIT CODE:
   - Open a C++, Python, or text file
   - Edit in main editor
   - Ctrl+S to save to disk
   - Syntax highlighting works for C++

================================================================================
ISSUES YOU MIGHT ENCOUNTER
================================================================================

TERMINAL TEXT HARD TO READ:
  - Try typing "cd d:\" then "dir" in terminal
  - If you see commands and output fine, it's working!

IDE DOESN'T RESPOND:
  - Check PowerShell terminal pane in bottom-left
  - If terminal is frozen, try pressing Ctrl+C

FILE SAVE FAILS:
  - Make sure file has a full path with extension (e.g., D:\test.cpp)
  - Can't save files without a name in the system

IDE CRASHES:
  - Look for RawrXD_Win32_IDE.exe in D:\rawrxd\Ship
  - Just re-run it to restart
