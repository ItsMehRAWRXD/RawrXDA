TROUBLESHOOTING GUIDE

ISSUE: "Cache corrupted" error
──────────────────────────────
Solution:
1. Delete C:\ProgramData\RawrXD\license.cache
2. Restart application
3. Re-activate license if needed

ISSUE: "License in grace period" warning
──────────────────────────────────────────
This is normal when:
• Network sync fails for extended period
• User operates offline beyond cache validity
Action: Reconnect to internet for online sync

ISSUE: "Tampering detected" alert
─────────────────────────────────
Critical: License key has been modified
Action:
1. Check audit log: C:\ProgramData\RawrXD\audit.log
2. Verify hardware HWID hasn't changed
3. Re-activate license if needed
4. Alert security team

ISSUE: Audit log rotation failure
──────────────────────────────────
Solution:
1. Check disk space (requires 1 MB minimum)
2. Verify permissions: C:\ProgramData\RawrXD\
3. Increase AUDIT_LOG_ROTATION_SIZE constant if needed
