## INTEGRATION STEPS

### 1. Compile
```bash
ml64.exe /c /O2 Phase4_Master_Complete.asm
```

### 2. Link as DLL
```bash
link /DLL /OUT:SwarmInference.dll Phase4_Master_Complete.obj kernel32.lib user32.lib vulkan-1.lib cuda.lib
```

### 3. Add to Win32IDEBridge
```cpp
// Header
#include "SwarmInference.h"

// In initialize()
swarm_master = SwarmInitialize(drive_paths);

// In main loop
ProcessSwarmQueue(swarm_master);

// On shutdown
SwarmShutdown(swarm_master);
```

### 4. Update CMakeLists.txt
```cmake
target_link_libraries(RawrXD-Win32IDE
    RawrXD-Win32IDEBridge
    RawrXD-AgenticFramework
    SwarmInference.lib  # ← Add this
)
```

### 5. Deploy
```bash
Copy SwarmInference.dll → C:\RawrXD\Framework\SwarmInference.dll
```
