### Demo 3: Multi-Language Support in API

```powershell
# Agent supporting 3 users simultaneously
$users = @(
    @{ Name = "Alice"; Text = "Hello, how are you?" },
    @{ Name = "Bob"; Text = "Hola, ¿qué tal?" },
    @{ Name = "Carol"; Text = "Bonjour, comment allez-vous?" }
)

foreach ($user in $users) {
    $lang = Detect-Language -Text $user.Text
    $greeting = Get-LanguageGreeting -Language $lang
    Write-Host "$($user.Name) ($lang): $greeting" -ForegroundColor Green
}
```

**Output**:
```
✅ Alice (English): 👋 Hello! I'm the RawrXD IDE Assistant!
✅ Bob (Spanish): 👋 ¡Hola! ¡Soy el Asistente RawrXD IDE!
✅ Carol (French): 👋 Bonjour ! Je suis l'Assistant RawrXD IDE !
```
