## 🔌 API QUICK REFERENCE

### Core Functions

```powershell
# Detect language from text
Detect-Language -Text "Your text here"

# Get all available languages
Get-AvailableLanguages

# Get greeting in specific language
Get-LanguageGreeting -Language 'Spanish'

# Create language session
Start-LanguageLearner -UserPreferredLanguage 'French'

# Update session with new input
Update-LanguageSession -Session $session -UserInput "New input"

# Get language statistics
Get-LanguageStats -Session $session

# Set user preference
Set-LanguagePreference -Session $session -Language 'German'

# Enable/Disable auto-detection
Enable-AutoLanguageDetection -Session $session
Disable-AutoLanguageDetection -Session $session
```

See `MULTI_LANGUAGE_SUPPORT_COMPLETE.md` for full API documentation.

## 💡 COMMON TASKS

### Task 1: Launch IDE with Multi-Language Support
```
1. Open PowerShell
2. Run: .\language_assistant.ps1
3. Click 🌍 Language button in IDE Browser
4. Select language from dropdown
5. Start typing!
```

### Task 2: Detect Language from User Input
```powershell
$input = "Bonjour, comment allez-vous?"
$language = Detect-Language -Text $input
Write-Host "Detected: $language"  # Output: "French"
```

### Task 3: Create Multi-Language Agent
```powershell
# Support multiple users in different languages
$englishSession = Start-LanguageLearner 'English'
$spanishSession = Start-LanguageLearner 'Spanish'
$frenchSession = Start-LanguageLearner 'French'

# Each maintains independent state
```

### Task 4: List All Supported Languages
```powershell
Get-AvailableLanguages | Format-Table Language, Native, Code
```

### Task 5: Get Language Statistics
```powershell
$stats = Get-LanguageStats -Session $session
$stats | ConvertTo-Json | Out-Host
```
