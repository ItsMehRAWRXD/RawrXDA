┌─────────────────────────────────────────────────────────────────────────────────┐
│ 💡 PRO TIPS                                                                      │
└─────────────────────────────────────────────────────────────────────────────────┘

   1. Backups are named with timestamps: index.html.backup.20251228-075221
   
   2. Multiple test runs create multiple reports: IDE-Test-Results-*.json
   
   3. Run tests while IDE is stopped to avoid port conflicts
   
   4. Check logs in the IDE console while testing (F12 in Electron)
   
   5. Use -Verbose flag for more detailed output
   
   6. You can test individual features (chat, models, etc.) without full suite
