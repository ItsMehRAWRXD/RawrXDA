# 🚀 Enhanced Desktop Copilot - Quick Start Card

## ⚡ 30-Second Start

```powershell
cd "D:\lazy init ide\desktop"
.\desktop_copilot_enhanced.ps1
```

**Done!** Copilot now in taskbar with 4 tabs of power.

---

## 📸 Media Viewer Tab

**View photos with metadata**

```
1. Click "📸 Media Viewer" tab
2. Drag image file → drop zone
3. See photo + file info
```

**Supported:** JPG, PNG, GIF, BMP, TIFF, WebP

---

## 📁 File Manager Tab

**Organize files automatically**

```
1. Click "📁 File Manager" tab
2. Drag folder → drop zone
3. Click "Analyze Files" → see breakdown
4. Click "Organize" → auto-sorted!
```

**Auto-creates:** Images, Documents, Videos, Programs, Archives, Other

---

## 🌐 Website Preview Tab

**Check websites before opening**

```
1. Click "🌐 Website Preview" tab
2. Enter URL
3. Click "📥 Fetch & Preview" → see content
4. Or click "🌐 Open" → launch browser
```

---

## 💻 System Info Tab

**Monitor your computer**

```
1. Click "💻 System Info" tab
2. See: CPU, RAM, Disk, Top Processes
3. Click "🔄 Refresh" → update anytime
```

## 💡 Example Workflows

### Clean Downloads
```
1. File Manager tab
2. Drag D:\Downloads
3. Analyze → see breakdown
4. Organize → creates D:\Downloads_Organized
5. Done! Files sorted by type
```

### View Vacation Photos
```
1. Media Viewer tab
2. Drag D:\Pictures\Vacation folder
3. See first photo
4. File info displayed
```

### Research a Website
```
1. Website Preview tab
2. Enter URL
3. Click Fetch & Preview
4. Read content summary
5. Click Open for full view
```
