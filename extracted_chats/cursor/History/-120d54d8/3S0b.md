## 🎯 **Quick Start**

### **Installation**
```bash
cd BigDaddyG-Standalone-IDE
npm install
```

### **Running the IDE**
```bash
# Windows
start.bat

# Linux/Mac
node server.js
```

### **Access the IDE**
Open your browser and navigate to: **http://localhost:3000**

## 🔧 **Usage Guide**

### **1. Project Management**
- Click **"New Project"** to create an Assembly project
- Projects include proper MASM structure with build scripts
- Automatic project scaffolding with templates

### **2. AI Model Selection**
- Choose from 5 specialized AI models in the sidebar
- Each model has unique capabilities for different tasks
- **Unrestricted execution** - no platform limitations

### **3. Code Development**
- Edit Assembly code in the main editor panel
- Syntax highlighting and formatting included
- Save files directly to the file system

### **4. Build & Run**
- **Build** projects using the toolbar buttons
- **Run** executables directly from the IDE
- Terminal output displayed in real-time

### **5. AI Assistance**
- Chat with AI models for programming help
- Get code examples and explanations
- Request code generation and optimization
