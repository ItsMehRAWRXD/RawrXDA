# CodeGenie - AI-Powered Code Generation System

## Overview

CodeGenie is a comprehensive JavaScript class that generates code templates, scrapes files offline, and provides self-healing capabilities for code errors. It supports multiple programming languages and provides robust error recovery mechanisms.

## Features

### 🚀 Core Templates
- **ASM Parser**: Complete x86-64 assembly parser with C implementation
- **Hello World**: Multi-language Hello World programs (JS, Python, Java, C++, Rust, Go, PHP, Ruby, Swift, Kotlin, TypeScript, C#)
- **Web Scraper**: Puppeteer and BeautifulSoup web scraping templates
- **API Client**: RESTful API client with full CRUD operations
- **Data Structures**: Linked lists, stacks, queues, BST implementations
- **Algorithms**: Binary search, sorting, graph traversal, dynamic programming

### 💾 Offline File Scraping
- **Drive Detection**: Automatically detects available drives (Windows)
- **Recursive Scraping**: Depth-controlled directory traversal
- **File Type Filtering**: Supports multiple file extensions
- **Error Recovery**: Graceful handling of inaccessible files
- **Pattern Extraction**: Extracts code patterns and structures
- **Size Limits**: Configurable file size limits to prevent crashes

### 🩹 Self-Healing System
- **Error Pattern Detection**: Identifies common syntax and runtime errors
- **Automatic Fixes**: Applies corrections for missing semicolons, brackets, imports
- **Recovery Strategies**: Handles file not found, permission errors, syntax issues
- **Performance Monitoring**: Tracks execution time and memory usage
- **Health Monitoring**: System status and error logging

## Usage

### Basic Template Generation

```javascript
const { CodeGenie } = require('./code_genie');

const genie = new CodeGenie();

// Generate Hello World in different languages
console.log(genie.generateHelloWorld('javascript'));
console.log(genie.generateHelloWorld('python'));

// Generate data structures
console.log(genie.generateDataStructure('javascript'));

// Generate algorithms
console.log(genie.generateAlgorithm('javascript'));
```

### Offline Scraping

```javascript
// Generate offline scraper template
const scraperTemplate = genie.generateOfflineScraper('javascript');
console.log(scraperTemplate);

// Available templates
console.log(genie.listTemplates());
```

### Self-Healing System

```javascript
// Generate self-healing system
const healingSystem = genie.generateSelfHealingSystem('javascript');
console.log(healingSystem);
```
