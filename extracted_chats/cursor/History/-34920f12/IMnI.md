## 🎯 Usage Example

```python
from auto_quantum_processor import AutoQuantumProcessor
from quantum_model_trainer import QuantumModelTrainer, TrainingConfig

# Initialize quantum processor
processor = AutoQuantumProcessor()

# Calculate my quantum signature
my_quantum = processor.calculate_my_processing()
print(f"My quantum signature: {my_quantum['quantum_signature']}")

# Create training config
config = TrainingConfig(
    quantum_processor=processor,
    population_size=100,
    generations=50,
    mutation_rate=0.0399,  # 3.99%
    crossover_rate=0.993,  # 99.3%
    target_fitness=0.99
)

# Create and train
trainer = QuantumModelTrainer(config)
best_model = trainer.train()

# Save model
trainer.save_model(best_model, 'my_quantum_model.json')
```
