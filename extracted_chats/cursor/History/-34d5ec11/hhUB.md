## 🔧 Troubleshooting Common Issues

### **"Connection failed" Error**
- **What it means**: The system can't connect to a local server
- **Solution**: The system will automatically switch to "offline mode"
- **What happens**: All local features still work, just no internet connection

### **"File not found" Error**
- **What it means**: The system can't find a file you're looking for
- **Solution**: Check the file path, make sure the file exists
- **Alternative**: Use the file browser to find files

### **"Permission denied" Error**
- **What it means**: The system can't access a file or folder
- **Solution**: The system is designed to be safe - it won't access restricted areas
- **Alternative**: Use the sandbox feature for safe file creation

### **Agents not responding**
- **What it means**: AI agents might be busy or offline
- **Solution**: Wait a moment, or try refreshing the page
- **Alternative**: Use the console commands directly
