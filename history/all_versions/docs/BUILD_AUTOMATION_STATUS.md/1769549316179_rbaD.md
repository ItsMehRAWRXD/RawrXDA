## 🚀 Usage

### One-Command Build & Wire
```batch
C:\RawrXD> Wire-RawrXD.bat
```

Or with PowerShell:
```powershell
C:\RawrXD> .\Wire-RawrXD.ps1
```

Both scripts:
1. Assemble all ASM components
2. Create static libraries
3. Link production executables
4. Wire artifacts to `C:\RawrXD`
5. Verify all outputs

### Run PE Generator
```batch
C:\RawrXD\bin> pe_generator.exe [options]
```

### Test Encoder
```batch
C:\RawrXD\bin> instruction_encoder_test.exe
```
