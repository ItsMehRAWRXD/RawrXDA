// 🐇 Tensor Bunny Hop [click to toggle]
Skip unneeded tensor layers during inference — lets bigger models run in smaller runtimes. The model "hops" through layers, keeping only the ones it needs for the answer.
Strategy auto
Skip Ratio 25%
Keep First N Layers 2
Keep Last N Layers 2
Custom Skip Layers
How it works:
When enabled, the inference request includes tensor_hop metadata. Layers not needed for the specific query are skipped, reducing VRAM/RAM usage and decode time. Auto mode analyzes the prompt to pick optimal skip patterns.

⚠️ Skip too many layers = quality drops. Start with 25% skip ratio.
