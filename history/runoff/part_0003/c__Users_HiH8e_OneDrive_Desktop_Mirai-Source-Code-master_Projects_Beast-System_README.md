# Beast System - Web-Based AI Automation Platform

## 🎯 Project Overview

**Beast System** is a comprehensive web-based AI automation platform for system control, swarm operations, and automated tasks.

### Key Features
✅ **Web Interface** - HTML5 dashboard  
✅ **Swarm Automation** - Coordinate multiple agents  
✅ **Python Backend** - Robust task execution  
✅ **Training Suite** - ML model training capabilities  
✅ **Demo System** - Interactive demonstrations  

---

## 🚀 Quick Start

### Using the Demo
```
Open: beast-swarm-demo.html in your browser
```

### Running the System
```powershell
python beast-mini-standalone.py
# or
python beast-swarm-system.py
```

### Training Models
```powershell
python beast-training-suite.py --data training_data.json
```

## 🏗️ Architecture

### Components

**Frontend**
- `beast-swarm-demo.html` - Interactive web interface
- `beast-swarm-web.js` - Web client logic

**Backend**
- `beast-swarm-system.py` - Core swarm orchestration
- `beast-mini-standalone.py` - Lightweight version
- `beast-training-suite.py` - Model training

**Tools**
- `Beast-IDEBrowser.ps1` - IDE integration
- `launch-beast-browser.ps1` - Quick browser launch
- `test-beast-performance.bat` - Performance validation

### Workflow
```
User → HTML Demo → JavaScript → Python Backend → Task Execution
                                    ↓
                            Model Training (optional)
```

## 🆘 Troubleshooting

### Demo Won't Load?
→ Check file path in browser
→ Ensure JavaScript is enabled

### Python Errors?
→ Check Python 3.8+ installed
→ Verify dependencies: See `beast-quick-start.py` for requirements

### Performance Issues?
→ Run: `test-beast-performance.bat`
→ Check system resources
→ Reduce agent count in configuration
