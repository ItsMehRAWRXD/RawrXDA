## 🔧 Core Functions to Use

### Logging (50-100 calls total)
```asm
LEA rcx, [rel msg_text]
MOV rdx, LOG_LEVEL_INFO              ; INFO=1, ERROR=3, etc.
CALL Logger_LogStructured
```

### Performance Measurement (20+ measurements)
```asm
CALL QueryPerformanceCounter
MOV r14, rax                         ; Save start time
; ... do work ...
CALL QueryPerformanceCounter
SUB rax, r14
CALL Metrics_RecordLatency           ; Record timing
```

### Engine Lifecycle (5 calls)
```asm
; Startup
CALL ZeroDayAgenticEngine_Create     ; rcx=router, rdx=tools, r8=planner, r9=callbacks
MOV [engine_handle], rax

; Shutdown
CALL ZeroDayAgenticEngine_Destroy    ; rcx=engine_handle
```

### Error Handling
```asm
CMP eax, 0
JE error_path
; success handling
LEA rcx, [rel msg_success]
MOV rdx, LOG_LEVEL_INFO
CALL Logger_LogStructured
```

## 🛑 Common Issues & Solutions

| Issue | Solution |
|-------|----------|
| "Unresolved external symbol" | Check masm_master_include.asm is included after .model |
| "Duplicate symbol definition" | Function defined in 2 files; use EXTERN only |
| Build takes forever | Incremental build; delete old .obj files |
| Performance overhead >5% | Reduce logging frequency, use sampling |
