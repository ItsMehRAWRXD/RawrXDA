- `cursor-ai-copilot.refactor` - Refactor selected code
- `cursor-ai-copilot.generate` - Generate code from prompt
- `cursor-ai-copilot.optimize` - Optimize selected code
- `cursor-ai-copilot.chat` - Open chat interface
- `cursor-ai-copilot.authenticate` - Authenticate with OpenAI
- `cursor-ai-copilot.logout` - Logout from extension
