## 💡 Tips & Tricks

### Maximize Productivity
1. Pin frequently used tabs to top
2. Use different models for different tasks
3. Enable reasoning only when needed (slower)
4. Keep terminal open for commands
5. Monitor health tab for resource bottlenecks

### Get Best Results
1. Be specific in chat prompts
2. Use reasoning for complex tasks
3. Compare multi-model outputs
4. Test browser agent on structured sites
5. Keep file explorer visible while coding

### Optimize Performance
1. Close unused tabs (>500 may slow down)
2. Restart terminals periodically
3. Monitor health metrics
4. Use lightweight models for fast response
5. Clear old chat history
