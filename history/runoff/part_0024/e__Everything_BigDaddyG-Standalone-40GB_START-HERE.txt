## 🆘 IF SOMETHING GOES WRONG

**Chat still not working after fix?**
```powershell
# Clean install
cd app
rm -r node_modules package-lock.json
npm install
npm start
```

**Ports already in use?**
```powershell
Get-Process node | Stop-Process -Force
# Wait 5 seconds
npm start
```

**Want to restore original?**
```powershell
Copy-Item "electron\index.html.backup.TIMESTAMP" "electron\index.html"
Copy-Item "electron\renderer.js.backup.TIMESTAMP" "electron\renderer.js"
```

**Need detailed diagnostics?**
```powershell
.\IDE-UI-Repair.ps1 -Diagnose
```
