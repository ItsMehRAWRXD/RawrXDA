INTEGRATION OVERVIEW:
─────────────────────

Main Loop:
   SwarmInitialize() → Get SWARM_MASTER*
   Loop {
       ProcessSwarmQueue(master)      ← Main scheduler
       UpdateMinimapHUD(master)       ← Already called from ProcessSwarmQueue
       Sleep(16)                      ← 60 FPS
   }
   SwarmShutdown(master)              ← Cleanup

Transport Control:
   SwarmTransportControl(master, command, parameter)
   • 0 = PLAY (forward @ 1x)
   • 1 = PAUSE (stop loading)
   • 2 = REWIND (backward @ 1x)
   • 3 = FF (fast forward, param = velocity)
   • 4 = STEP (advance 1 episode)
   • 5 = SEEK (jump to episode, param = index)

I/O Architecture:
   JBOD (11TB)
   ├── 4× NVMe 4TB each
   └── 1× SATA 1TB
        ↓ NTFS MFT scan
   Tensor Catalog (1M entries)
        ↓ DMA + sparsity filtering
   Fabric (1.6TB virtual MMF)
        ↓ GPU binding
   VRAM (40GB hot episodes)

Performance Specs:
   • Per-drive throughput: 300-500 MB/s
   • Aggregate throughput: 1.5-2.5 GB/s (5 drives parallel)
   • Episode load time: <2ms (512MB per episode)
   • Queue latency: <10ms
   • Inference dispatch: <10µs
   • Sparsity savings: 50-95% bandwidth reduction
   • VRAM usage: 0-40GB (adaptive)
