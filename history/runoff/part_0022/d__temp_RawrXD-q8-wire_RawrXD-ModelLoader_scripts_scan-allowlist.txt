# 
# Each non-empty, non-comment line is a regex pattern.
# Patterns match against "file:line: text" or "file: text" format.
# Use this to suppress known TODOs, placeholders, or missing definitions
# to keep CI builds green while tracking technical debt separately.
#
# Examples:
#
# Suppress all TODOs in a specific file:
# MainWindow\.cpp:.*TODO
#
# Suppress a specific TODO by line content:
# .*TODO.*implement streaming.*
#
# Suppress missing definitions for a specific class/method:
# src\\qtapp\\.*: .*ProjectExplorerWidget::.*
#
# Suppress empty bodies in test files:
# tests\\.*: .*\{\s*\}
#
