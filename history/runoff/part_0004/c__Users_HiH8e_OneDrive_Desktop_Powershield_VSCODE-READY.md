## 🚀 Quick Start (3 Steps)

### 1. Reload VS Code
```
Ctrl+Shift+P → Developer: Reload Window
```

### 2. Start Ollama
```bash
ollama serve
```

### 3. Toggle Agentic Mode
```
Press Ctrl+Shift+A
```

You'll see **🚀 AGENTIC** in the bottom right corner (green)

**Completions not working?**
- Make sure agentic mode is ON (Ctrl+Shift+A)
- Check Ollama is running
- Check settings (Ctrl+,) → search "agentic"
