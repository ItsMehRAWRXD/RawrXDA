GETTING STARTED
================================================================================

1. LAUNCH THE APPLICATION
   Windows: bin\Release\RawrXD-Agentic-IDE.exe
   Linux:   ./bin/RawrXD-Agentic-IDE
   macOS:   ./bin/RawrXD-Agentic-IDE.app/Contents/MacOS/RawrXD-Agentic-IDE

2. MAIN INTERFACE OVERVIEW
   - Menu Bar: File, Edit, View, Tools, Help
   - Toolbar: New Canvas, New Chat, Save, Open, About
   - Central Area: 3 main tabs (Paint, Chat, Code)
   - Dock Widgets: Features Panel, Statistics, Output Console

3. PAINT EDITOR
   - Create: Click "+" button or press Ctrl+N
   - Draw: Select tool from toolbar
   - Save: Ctrl+S or File > Save
   - Export: PNG or BMP format
   - Features: Unlimited tabs, undo/redo, real-time rendering

4. CHAT INTERFACE
   - New Chat: Ctrl+Shift+N or click "+" button
   - Send: Type message and press Enter
   - Multiple Chats: Switch between 100+ concurrent sessions
   - Close: Click "×" button or Ctrl+W

5. CODE EDITOR
   - Open: File > Open or Ctrl+O
   - MASM Mode: Optimized for 1M+ lines
   - Standard: 100K+ lines for other languages
   - Features: Syntax highlighting, lazy loading, tab pooling

6. FEATURES PANEL
   - Toggle: Ctrl+F to show/hide
   - Search: Type to filter features
   - Enable/Disable: Right-click on feature
   - Categories: Drawing, Export, Chat, Code, UI, Plugins, Performance

PAINT EDITOR TOOLS
================================================================================

Tool Selection:
- P: Pencil (precise pixel-perfect strokes)
- B: Brush (soft, anti-aliased strokes)
- E: Eraser (removes pixels)
- L: Line (straight lines)
- R: Rectangle (filled or outline)
- C: Circle (filled or outline)
- Y: Polygon (multi-point shapes)
- F: Fill Bucket (area fill with color)
- K: Color Picker (sample colors)

Quick Actions:
- Ctrl+Z: Undo
- Ctrl+Y: Redo
- Ctrl+S: Save canvas
- Ctrl+Shift+S: Save all canvases
- Ctrl+N: New canvas tab
- Ctrl+W: Close current tab

Brush Settings:
- Brush Size: Use toolbar slider (1-100 pixels)
- Opacity: Use opacity slider (0-100%)
- Color: Click color button to open color picker

================================================================================
COMMON TASKS
================================================================================

TASK: Draw and Export a Painting
1. Create new tab (Ctrl+N)
2. Select drawing tool (P for pencil, B for brush)
3. Draw on canvas
4. Click Save or press Ctrl+S
5. Choose PNG or BMP format
6. Select location and filename

TASK: Have Multiple Conversations
1. Click "+" in Chat tab or press Ctrl+Shift+N
2. Type in first conversation
3. Click "+" again for second conversation
4. Switch between conversations using tabs
5. Each maintains separate history

TASK: Edit Large Code Files
1. Open file (Ctrl+O)
2. Select MASM mode for large assembly files
3. Use Ctrl+G to navigate to lines
4. Use Ctrl+F for searching
5. Syntax highlighting updates in real-time

TASK: Customize Features
1. Press Ctrl+F to show Features panel
2. Search for feature you want
3. Right-click on feature
4. Select Enable or Disable
5. Changes saved automatically

================================================================================
TIPS & TRICKS
================================================================================

1. Save your work frequently
   - Use Ctrl+S for current tab
   - Use Ctrl+Shift+S for all tabs

2. Organize your work
   - Use meaningful tab names
   - Group related paintings/chats
   - Close unused tabs to save memory

3. Use keyboard shortcuts
   - Most actions have shortcuts
   - Faster than mouse navigation
   - Customizable in settings (future)

4. Monitor performance
   - View Statistics panel (right dock)
   - Shows tab counts and usage
   - Helps optimize workflow

5. Export frequently
   - Paintings: PNG for quality, BMP for compatibility
   - Regular exports prevent data loss
   - Multiple formats available

6. Dark theme benefits
   - Reduces eye strain during long sessions
   - Professional appearance
   - Better performance on high DPI displays

================================================================================
TROUBLESHOOTING
================================================================================

Issue: Application won't start
Solution: Ensure Qt6 libraries are installed and in system path

Issue: Icons not showing
Solution: Verify resources.qrc is compiled and resources.rcc exists

Issue: Slow performance
Solution: Close unused tabs, check Statistics panel for memory usage

Issue: Paint doesn't export
Solution: Ensure write permissions, try different location or format

Issue: Features panel is empty
Solution: Check that features are registered in code (11 default)

Issue: Code editor not showing syntax highlighting
Solution: Verify file extension and check Code Editor settings
