================================================================================
FILE OPERATIONS
================================================================================

Ctrl+N          New Paint Canvas
                Create a new blank paint canvas in a new tab

Ctrl+Shift+N    New Chat
                Start a new chat conversation in a new tab

Ctrl+O          Open File
                Open an existing file (image or code)

Ctrl+S          Save Current
                Save/export the current paint canvas

Ctrl+Shift+S    Save All
                Save/export all paint canvases

Ctrl+W          Close Current Tab
                Close the active tab with unsaved change warning

Alt+F4          Exit Application
                Ctrl+Q also works

================================================================================

Ctrl+Y          Redo
                Redo last undone action in paint editor

================================================================================
PLATFORM-SPECIFIC NOTES
================================================================================

WINDOWS:
- All shortcuts use Ctrl (not Cmd)
- Alt key accesses menu bar
- Alt+F4 closes application
- Windows key opens OS menu

LINUX:
- All shortcuts use Ctrl
- Alt key behavior depends on window manager
- Middle mouse click pastes (many distributions)
- Alt+Tab switches between applications

macOS:
- Use Cmd instead of Ctrl
- Option key replaces Alt
- Cmd+Q quits application
- Cmd+Tab switches applications

================================================================================
TROUBLESHOOTING SHORTCUTS
================================================================================

Shortcut Not Working?
1. Verify correct key combination
2. Check if feature is enabled
3. Ensure focus is in correct widget
4. Try closing/reopening tab
5. Restart application

Conflicting Shortcuts?
- Some OS shortcuts may override IDE shortcuts
- Check OS keyboard settings
- Modify OS shortcuts if needed
- Contact support if persistent

Forgotten Shortcuts?
- This reference document lists all shortcuts
- Tooltips appear on hover in UI
- Help menu contains keyboard reference
- Check documentation files
