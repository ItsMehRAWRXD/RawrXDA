## 📞 TROUBLESHOOTING

**Problem**: "Unresolved external symbol"  
**Solution**: Check if method is in .h file AND has body in .cpp

**Problem**: "Method does not name a type"  
**Solution**: Check declaration in header before implementation

**Problem**: Application still crashes  
**Solution**: Check for nullptr pointers (m_fileBrowserDock, etc.)

**Problem**: Menu item doesn't work  
**Solution**: Verify signal/slot connection (already done, check logic)

**Problem**: "Cannot find file chat_interface.h"  
**Solution**: Use correct include path (src/ vs include/)
