Full-featured Electron UI with ALL capabilities enabled. No restrictions, no placeholders - everything is functional and ready to use.

## Requirements
- Node.js 18+
- Windows, macOS, or Linux

## Install
```powershell
npm install
```

## Run (development)
```powershell
npm run dev
```

This starts Electron and opens the enhanced dark-themed UI.
