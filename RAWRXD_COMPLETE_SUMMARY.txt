1. Install Visual Studio 2022 with:

   - Windows 10/11 SDK
   - C++ CMake tools (optional)

2. Open x64 Native Tools Command Prompt

3. Navigate to RawrXD directory:
   cd D:\RawrXD

4. Run build script:
   build_master.bat

5. Output in .\bin\:
   RawrXD_IDE.exe
   RawrXD_CLI.exe
   RawrXD_Inference.dll
   RawrXD_Deobfuscator.dll

6. Run:
   .\bin\RawrXD_IDE.exe

   .\bin\RawrXD_Launcher.bat
